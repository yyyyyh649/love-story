# 💕 恋爱日记本 - Love Story Diary

这年头谁还记录谈恋爱啊..... 我们还记录！

## 📖 项目简介

恋爱日记本是一个简单而优雅的网页应用，帮助你记录和珍藏与爱人的每一个美好瞬间。无论是第一次约会、浪漫的旅行，还是平凡的日常，都值得被记录下来。

## ✨ 功能特性

### 📝 记录日记
- 添加日期、标题、地点、事件等详细信息
- 支持长文本内容记录
- 支持上传多张图片
- 实时图片预览

### 🔍 智能查找
- **按日期查找**：快速找到某一天的日记
- **按地点查找**：查看在某个地方发生的所有故事
- **按事件查找**：搜索特定事件的记录
- 支持组合筛选和清除筛选

### 📤 导出功能
- **导出为PDF**：使用浏览器打印功能将日记保存为PDF
- **导出为PNG**：将日记导出为图片格式
- 支持单条日记导出或全部导出
- 导出时自动隐藏操作按钮，保持内容整洁

### 💾 本地存储
- 使用浏览器本地存储（LocalStorage）
- 无需服务器，数据完全保存在本地
- 隐私安全，只有你能看到

### 🎨 精美设计
- 温馨浪漫的渐变配色
- 响应式设计，支持手机、平板、电脑
- 流畅的动画效果
- 直观友好的用户界面

## 🚀 如何使用

### 方式一：直接打开（推荐）
1. 克隆或下载本仓库
2. 直接在浏览器中打开 `index.html` 文件
3. 开始记录你们的故事！

### 方式二：使用本地服务器
```bash
# 克隆仓库
git clone https://github.com/yyyyyh649/love-story.git
cd love-story

# 使用 Python 启动简单服务器
python -m http.server 8000

# 或使用 Node.js
npx http-server

# 然后在浏览器打开 http://localhost:8000
```

## 📱 使用指南

### 添加日记
1. 选择日期（默认为今天）
2. 填写标题（必填）
3. 填写地点和事件（可选，用于后续查找）
4. 记录今天的故事（必填）
5. 可选：上传图片留住美好瞬间
6. 点击"保存日记"按钮

### 查找日记
1. 使用顶部的筛选器：
   - 选择日期查看该日的日记
   - 输入地点关键词（如"星巴克"）
   - 输入事件关键词（如"约会"）
2. 筛选器会实时更新显示结果
3. 点击"清除筛选"恢复显示所有日记

### 导出日记
1. **导出为PDF**：
   - 点击"导出PDF"按钮
   - 浏览器会打开打印对话框
   - 选择"保存为PDF"
   - 选择保存位置

2. **导出为PNG**：
   - 点击"导出PNG"按钮
   - 图片会自动下载到下载文件夹

### 删除日记
- 点击每条日记下方的"删除"按钮
- 确认后即可删除（无法恢复，请谨慎操作）

## 💡 使用建议

1. **定期备份**：虽然数据保存在本地很安全，但建议定期导出重要日记作为备份
2. **图片大小**：上传的图片会转换为Base64存储，建议适当压缩图片以节省空间
3. **浏览器兼容**：建议使用现代浏览器（Chrome、Firefox、Edge、Safari最新版）
4. **隐私保护**：数据仅保存在你的浏览器中，清除浏览器数据会删除所有日记
